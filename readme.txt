== Description ==

We are going to enable the WordPress ping service for each site in a WordPress MU Network. This allows each site to have a different ping list. By default, you have no ping list for WordPress MU sites.

= New =

* Nothing

= Features =

* Enables the custom ping for WordPress sites in WP MU.
* Individual site settings for WPMU.


== Installation ==

Follow the steps below to install the plugin.

1. Upload the Site Ping Plugin directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to <a href="options-writing.php">Settings -> Writing</a> to update the ping list.
